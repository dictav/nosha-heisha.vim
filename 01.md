### Visual モード

* `v`: 文字指向(`o`でトグル)
* `V`: 行指向
* `<C-v>`: ブロック指向

#### 小文字を大文字に

```
<li>one</li>
<li>two</li>
<li>three</li>
<li>four</li>
```

#### 行末にセミコロン

```
var a = 1
var str = "string"
var hoge = 1 + 234 + 567
```

#### 余分なスペースを削除

```
Chapter                 Page 
Normal mode              1
Insert mode              2
Visual mode              3 
```

#### ２列目と３列目を入れ替え

```
Chapter        Page    Amount    
Normal mode     1         10
Insert mode     2         20
Visual mode     3         30
```
